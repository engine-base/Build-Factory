# データ分析（GA4・Search Console）

## このスキルがやること
GA4・Search Consoleのデータを定期的に読み込み、流入・行動・CV・異常を検知してレポート化する。
「数字は見てるけど何も判断できていない」状態から脱出するための翻訳・解釈スキル。

---

## 週次確認（月曜 15分）

| 指標 | ツール | 確認内容 |
|------|--------|---------|
| セッション数 | GA4 | 先週比・異常な増減 |
| 流入源別 | GA4 | Organic / Direct / SNS / Referral の比率 |
| コンバージョン数 | GA4 | 問い合わせ・資料DL等 |
| 検索クリック数 | Search Console | 前週比 |
| インデックス状態 | Search Console | エラー・警告 |

**アウトプット：** 週次スナップショット（5行サマリ + 要対応事項）

---

## 月次レポート（月初 45〜60分）

**インプット：** 前月の GA4・Search Console エクスポートデータ

**アウトプット：**
```
月次Webパフォーマンスレポート
─────────────────────
■ 流入サマリ
  - 総セッション: ○○件（前月比 +○%）
  - オーガニック: ○○件 / SNS: ○○件 / Direct: ○○件

■ コンバージョン
  - 問い合わせ: ○件（前月比 ±○）
  - CVR: ○○%

■ 上位流入ページ Top5

■ 検索パフォーマンス
  - 表示回数: ○○万回 / クリック数: ○○件 / CTR: ○○%

■ 今月の異常・発見事項

■ 来月の重点改善アクション
```

---

## 異常検知のトリガー

以下が発生したら即座にアラート出力：
- セッション数が前週比 -30%以上
- 特定ページのインデックスが外れた
- CVが先月比で半分以下
