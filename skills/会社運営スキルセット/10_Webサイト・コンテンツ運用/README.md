## write-article.skill との関係

```
既存スキル: write-article.skill（記事の執筆・生成）
      ↓ 作成した記事を
このカテゴリで管理・運用
      ├── 記事管理・コンテンツ監査（パフォーマンス追跡・更新判断）
      ├── SEO定点観測・レポート（記事の検索順位チェック）
      └── データ分析（流入数・離脱率の確認）
```

**write-article.skillはこのカテゴリの「コンテンツ生産ライン」として位置づけ。**
記事を作る → このカテゴリで育てる・管理する、という役割分担。

---

## 週次・月次の運用サイクル

```
【週次（月曜 30分）】
1. SEO定点観測：順位変動・流入チェック → 異常あれば即アクション
2. GA4確認：先週のPV・CV数確認
3. メールマーケティング：配信スケジュール確認

【月次（月初 2時間）】
1. SEOレポート作成：前月比・競合比較・改善提案
2. コンテンツ監査：更新・削除・新規記事の優先順位更新
3. LP改善：CVR確認・次月のA/Bテスト設計
4. データ分析：月次サマリレポート生成
5. メール：前月の開封率・CTR分析→次月の改善
```
