# SEO定点観測・レポート

## このスキルがやること
HP・LP・ブログの検索順位変動・オーガニック流入・クロール状態を週次・月次で確認し、
現状レポートと改善指示を自動生成する。
「SEOやってるつもりで何も変わらない」を防ぐ数値ドリブンな運用ループ。

---

## 運用サイクル

### 週次チェック（毎週月曜 / 所要15〜30分）

**インプット**
- Search Console（過去7日の検索パフォーマンス）
- 順位チェックツールのデータ（前週比）
- GA4（セッション数・流入源）

**アウトプット**
- 週次ダッシュボード更新（順位変動・流入数・CTR）
- 急落キーワードのアラートリスト
- 即対応が必要な項目の優先アクションリスト

---

### 月次レポート（月初 / 所要60〜90分）

**インプット**
- 前月の Search Console 全データ
- GA4 月次データ
- 競合サイトの動向（オプション）

**アウトプット**
- 月次SEOレポート（下記構成）
  - サマリ（前月比：順位・流入・CTR・表示回数）
  - キーワード別パフォーマンス表（上昇・下降・新規）
  - 上位流入記事・LP一覧
  - インデックス・クロール状態
  - 競合との比較（設定している場合）
  - 翌月の改善アクションリスト（優先度付き）

---

## 改善策の出力フォーマット

```
【優先度: 高】
- キーワード「XXX」が先月比-8位。タイトルと見出しを見直す。
  → 対象記事: /blog/xxx
  → 推奨対応: H1に「XXX」を含める・冒頭200字に追加
  → 期限: 今月中

【優先度: 中】
- CTRが低いページ（表示多・クリック少）が3ページ。
  → メタディスクリプションを書き直す
```

## write-article.skillとの連携
改善アクションの中に「新規記事が必要」と判断されたものは
write-article.skillへトスアップし、記事生成→公開→翌月の観測へループする。
