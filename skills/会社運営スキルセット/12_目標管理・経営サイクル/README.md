# 12_目標管理・経営サイクル

**役割：** 会社の全活動を「どこに向けて動かすか」を決め、定期的に確認・修正するリズムを作る。
これがないと、個々のスキルが揃っていても「頑張っているが、何に向かっているかわからない」状態になる。

**このカテゴリが他の全カテゴリの「上位レイヤー」。**

---

## なぜ単独カテゴリが必要か

05_経営戦略（市場調査・事業計画）は「方向を考える」ための道具。
このカテゴリは「方向を決めて、実際に全業務をそこに向けて動かし続けるサイクル」。別物。

```
05_経営戦略（考える・設計する）
      ↓ 出力
12_目標管理・経営サイクル（決める・動かす・修正する）← ← 毎週・毎月・毎四半期ループ
      ↓ 優先事項を渡す
  01〜11の全カテゴリ（実行する）
```

---

## 目標の階層（カスケード構造）

```
【ビジョン】  3〜5年後になりたい姿（年1回設定）
      ↓
【年次OKR】  今年達成すること（年初に設定・年末に評価）
      ↓
【四半期OKR】 この3ヶ月の重点（四半期初に設定・末に評価）
      ↓
【月次目標】  今月の3つの優先事項（月初設定・月末振り返り）
      ↓
【週次優先事項】 今週やること（月曜設定・金曜レビュー）
      ↓
【日次Top3】  今日の3タスク（毎朝設定）← 13_セルフマネジメントと連携
```
