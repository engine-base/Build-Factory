# 月次振り返り・目標チェック

## このスキルがやること
月末または月初に、先月の実績を振り返り、今月の3つの優先事項を設定する。
四半期OKRと日々の業務を繋ぐ「月次の幹」。

---

## 実施タイミング
- **毎月最終営業日 or 翌月第1営業日**（月1回）
- 所要時間：60〜90分

---

## 月次レビューの構成

### Part 1：先月の振り返り（30分）

**定量チェック（数値で見る）**
```
売上・入金:    ○○円（目標比 ○○%）
問い合わせ数:  ○件
受注数:       ○件
Webセッション: ○○件（前月比）
SNS反応:      主要投稿のエンゲージメント
```

**定性チェック（言葉で見る）**
- 先月やって良かったこと（続けること）
- 先月うまくいかなかったこと（やめること・改善すること）
- 予定外に発生したこと・気づき

**四半期OKRとの照合**
```
KR 1-1: 進捗 ○○%（予定ペース: ○○%）→ 順調 / 遅れ / 停止
→ 遅れている場合の回復アクション
```

### Part 2：今月の目標設定（30分）

**今月の最優先事項 Top3**
```
優先事項 1: ○○○（達成基準: ○○○）
優先事項 2: ○○○（達成基準: ○○○）
優先事項 3: ○○○（達成基準: ○○○）
```

**今月「絶対やらないこと」**
（先月の振り返りから来るもの）

**今月の各カテゴリへの指示**
```
10_Web運用: 今月の重点改善箇所 → ○○
01_営業: 今月のアウトリーチ目標 → ○件
03_財務: 今月の注意点 → ○○
```

### Part 3：アウトプット

- 月次振り返りレポート（自分用記録）
- 今月のTop3タスクカード（見える場所に貼る）
- 各カテゴリへの優先事項の反映指示

---
