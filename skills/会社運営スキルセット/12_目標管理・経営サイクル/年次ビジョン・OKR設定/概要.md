# 年次ビジョン・OKR設定

## このスキルがやること
年末〜年初にかけて、3〜5年のビジョンを確認し、今年1年のOKR（目標と主要な結果）を設定する。
「なんとなく今年も頑張る」から「この年が終わった時に何が変わっているべきか」を決める。

---

## 実施タイミング
- **毎年12月下旬〜1月上旬**（年に1回）
- 所要時間：半日〜1日（思考を伴うため分割してもよい）

---

## インプット
- 前年の年次OKR達成状況（四半期レビューのアウトプット）
- 現在の財務状況（03_財務・経理のP&L年次版）
- 市場・競合環境の変化（05_経営戦略/市場調査のアウトプット）
- 自分の「やりたいこと・やりたくないこと」の棚卸し

---

## アウトプット

### 1. ビジョン確認・更新（3〜5年）
```
【ビジョン】2028年に自分はどんな状態か？
- 年商: ○○円
- 事業の形: ○○（受託 / SaaS / 両立 等）
- チームの状態: ○○
- 自分の働き方: ○○
```

### 2. 年次OKR（今年の3〜5つの目標）
```
Objective 1: ○○○（定性的な目標）
  Key Result 1-1: ○○○（数値で測れる結果）
  Key Result 1-2: ○○○
  Key Result 1-3: ○○○

Objective 2: ○○○
  Key Result 2-1: ○○○
  ...
```

### 3. 今年の「やらないこと」リスト
目標と同じくらい重要。何を断るかを先に決める。

### 4. 四半期への分解
年次OKRをQ1〜Q4に割り振る（初期案）

---

## OKR設計の原則
- Objectiveは「刺激的で方向感がある」こと（数値ではなく言葉）
- Key Resultは「数値で達成を判断できる」こと
- 達成確率70%程度の難易度が最適（100%確実なものはOKRではない）
- 1Objectiveにつき2〜4 Key Resultが目安
