# 四半期レビュー・OKR更新

## このスキルがやること
3ヶ月ごとに、前四半期のOKR達成状況を評価し、次四半期のOKRを設定する。
年次OKRという「大きな方向」を、3ヶ月という「実行できる単位」に落とし込む橋渡し。

---

## 実施タイミング
- **3月末・6月末・9月末・12月末**（年4回）
- 所要時間：2〜3時間
- 場所：できれば普段と違う環境で（カフェ・図書館等）

---

## インプット
- 前四半期のOKR（設定書）
- 前四半期の実績数値（財務・KPI・プロジェクト完了状況）
- 10_Webサイト・コンテンツ運用の月次データ（3ヶ月分）

---

## アウトプット

### 1. 前四半期のOKR評価
```
Objective 1: ○○○
  KR 1-1: 目標100 → 実績78 → 達成率78%
  KR 1-2: 目標○○ → 実績○○ → 達成率○○%
  総評: 達成 / 未達 / 超過達成

【未達の場合】根本原因分析：
  - 見積もりが甘かった / 外部要因 / 優先度が変わった
  → 次四半期への教訓
```

### 2. 次四半期のOKR設定
```
Q○ Objective 1: ○○○
  KR 1-1: ○○○（数値目標）
  KR 1-2: ○○○

【年次OKRとの整合性チェック】
年次OKRに対してこの四半期が貢献するか確認
```

### 3. 四半期の「重点テーマ」（1〜2個）
OKRとは別に、この3ヶ月で特に集中したいことを宣言

### 4. 各カテゴリへの優先度指示
```
01_営業・集客: この四半期は○○に注力
03_財務・経理: 要注意ポイント○○
10_Web運用: 重点キーワード○○
```
