# 09_情報・ナレッジ管理

**役割：** 日々の学びと経験を会社の資産に変える。
一人会社の最大の弱点は「自分が止まると全てが止まる」こと。知識を外部化・構造化することで、AIが代替できる範囲を広げる。

---

## スキル一覧

| スキル | やること |
|--------|---------|
| [ナレッジベース構築](./ナレッジベース構築/) | 社内ドキュメントの整理・検索設計・情報の一元管理設計 |
| [情報収集・キュレーション](./情報収集・キュレーション/) | 業界ニュース・技術情報の自動収集・要約・配信設計 |
| [学習記録・スキルマップ](./学習記録・スキルマップ/) | 習得スキル・知識ギャップの可視化と学習計画の生成 |

---

## 前後のつながり

```
全ての業務から得た知見
      ↓
ナレッジベース構築（蓄積）
      ↓
情報収集・キュレーション（インプット）
      ↓
学習記録・スキルマップ（成長の可視化）
      ↓
より良いスキル・判断 → 全業務へフィードバック
```
