# 08_カスタマーサクセス

**役割：** 納品後の顧客を離さず、追加受注・継続契約につなげる。
新規獲得より既存継続の方がコストが低い。CSは最高の営業活動。

---

## スキル一覧

| スキル | やること |
|--------|---------|
| [オンボーディング設計](./オンボーディング設計/) | 納品直後の導入支援計画・操作マニュアル・スタートアップガイドの生成 |
| [サポート対応](./サポート対応/) | 問い合わせ内容を分析し、回答ドラフトを生成。FAQの自動整備 |
| [顧客満足度計測](./顧客満足度計測/) | NPS・満足度アンケートの設計・結果分析・改善提案の生成 |
| [アップセル設計](./アップセル設計/) | 追加提案のタイミング設計・提案書の自動生成（次フェーズ・保守契約等） |

---

## 前後のつながり

```
delivery.skill（納品）← 開発OS
      ↓
オンボーディング設計（使い始めを支援）
      ↓
サポート対応 + 顧客満足度計測（継続フォロー）
      ↓
アップセル設計 → hearing.skill（次案件の商談）
```

## 優先度
- ★★★ 今すぐ：オンボーディング設計
- ★★☆ 3ヶ月以内：サポート対応、顧客満足度計測、アップセル設計
