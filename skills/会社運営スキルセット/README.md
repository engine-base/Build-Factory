## 会社を動かす全体像

```
【年次】ビジョン・OKR設定（12）
  ↓
【四半期】OKRレビュー・重点テーマ決定（12）
  ↓
【月次】目標チェック・振り返り（12）← データは全カテゴリから
  ↓
【週次】レビュー・来週の優先設計（12）
  ↓
【日次】Top3タスク・時間設計（13）
  ↓
【実行】01〜11の各カテゴリ（業務の実行）
         ↘                    ↙
         開発OS 14フェーズ（受注〜納品）
  ↓
【守り】14リスク管理 / 15ネットワーク管理（継続のための土台）
```
