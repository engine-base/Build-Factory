# 13_セルフマネジメント

**役割：** 一人会社において「自分」は最重要かつ唯一の経営資源。
自分の生産性・エネルギー・集中力を最大化することが、そのまま会社の最大化になる。
「忙しいのに前に進んでいる感じがしない」を根本から解決するカテゴリ。

---

## なぜ単独カテゴリが必要か

12_目標管理・経営サイクルが「会社レベルの羅針盤」なら、
このカテゴリは「自分レベルの動かし方」。

目標が決まっても、日々のタスク管理・時間設計・習慣がなければ実行されない。
一人会社では「マネージャーが自分を管理してくれる」は存在しないため、自己設計が必須。
