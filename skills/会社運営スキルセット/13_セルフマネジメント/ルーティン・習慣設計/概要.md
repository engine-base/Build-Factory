# ルーティン・習慣設計

## このスキルがやること
朝・夜・週の定型行動（ルーティン）を設計し、毎回「何をしようか」と考えるコストをゼロにする。
意志力は有限。ルーティン化することで判断疲れを防ぎ、重要な決断に使えるエネルギーを温存する。

---

## 設計する3つのルーティン

### 朝のルーティン（起床〜仕事開始 30〜60分）
```
例：
□ 起床 → 水を飲む
□ 軽い運動 or 散歩（15分）
□ 今日のTop3タスクを設定（タスク管理スキルと連携）
□ 経営ダッシュボードを30秒確認
□ 仕事開始
```

**ポイント：**
- SNS・メールは朝のルーティンが終わるまで開かない
- 「仕事開始の儀式」を決める（特定の音楽・コーヒーを淹れる等）

### 夜のルーティン（仕事終了〜就寝 15〜30分）
```
例：
□ 今日のTop3の達成確認
□ 未完タスクの明日への引き継ぎメモ
□ 明日の最初のタスクをカレンダーに入れる
□ 仕事終了宣言（PCを閉じる等の儀式）
```

**ポイント：**
- 「仕事終了の儀式」が重要。在宅の場合は特に「切り替え」が必要

### 週次ルーティン（金曜午後 または 月曜朝）
→ 12_目標管理・経営サイクル/週次レビューと連携

---

## ルーティン設計のインプット
- 自分の集中できる時間帯（朝型 or 夜型）
- 固定の制約（子どもの送迎・固定MTG等）
- 現在できていること・できていないこと

## アウトプット
- 自分専用の朝/夜ルーティンシート
- 習慣トラッカー（○×で継続確認）
- 1ヶ月後の振り返りフォーマット（継続できたか・効果があったか）
