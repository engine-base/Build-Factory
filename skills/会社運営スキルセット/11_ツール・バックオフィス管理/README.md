# 11_ツール・バックオフィス管理

**役割：** 会社の「裏側」を効率化する。
ツール費用の最適化・評判管理・受信ボックスの整理・料金設計など、
目立たないが放置すると地味にコストと機会損失が積み上がる領域。

---

| スキル | やること |

| [口コミ・評判管理](./口コミ・評判管理/) | Google・SNS・クラウドソーシングの評価収集と返信・改善 |
