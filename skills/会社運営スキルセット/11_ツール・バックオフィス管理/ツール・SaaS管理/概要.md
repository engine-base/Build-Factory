# ツール・SaaS管理

## このスキルがやること
現在使っているすべてのツール・SaaS・サブスクリプションを一元管理し、
コスト最適化・不要サービスの解約・更新タイミングの把握を行う。
「気づいたら月○万円ツール代に消えていた」を防ぐ。

---

## ツール台帳の項目

| 項目 | 内容 |
|------|------|
| ツール名 | - |
| カテゴリ | SEO / 開発 / 請求 / コミュニケーション 等 |
| 料金 | 月額 or 年額 |
| 支払い方法 | カード / 振込 |
| 更新日 | アラートを設定 |
| 活用度 | 毎日 / 週1 / ほぼ未使用 |
| 代替可能か | 有料→無料代替の有無 |
| 解約難易度 | 簡単 / 要連絡 |

---

## 月次レビュー（月初 15分）

**確認事項：**
- 今月の更新・課金予定ツール一覧
- 活用度が低いツールの解約検討
- 新たに導入したツールの登録

**アウトプット：**
- ツールコスト月次サマリ（カテゴリ別合計）
- 解約推奨リスト（理由付き）
- 03_財務・経理への経費連携データ

---

## 四半期コスト監査（3ヶ月に1回 45分）

- 全ツールの活用度再評価
- 類似ツールの統合・重複排除
- 年間更新 vs 月次契約の見直し
- 新技術への切り替え検討（AIツールは特に進化が早い）
