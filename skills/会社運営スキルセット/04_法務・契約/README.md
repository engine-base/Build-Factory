# 04_法務・契約

**役割：** 法的リスクから会社を守る。
契約ミスは致命傷になる。条件の抜け・不利条項・口頭合意の放置を全てAIで防ぐ。

---

## スキル一覧

| スキル | やること |
|--------|---------|
| [業務委託契約書作成](./業務委託契約書作成/) | 条件（報酬・納期・著作権・責任範囲）を入力→契約書ドラフトを即生成 |
| [NDA作成・レビュー](./NDA作成・レビュー/) | 秘密保持契約のドラフト生成と、相手方NDAのリスクチェック |
| [利用規約・プライバシーポリシー作成](./利用規約・プライバシーポリシー作成/) | SaaS・受託サービス向けの利用規約とPP（個人情報保護方針）の生成 |
| [契約書リスクレビュー](./契約書リスクレビュー/) | 相手方から受け取った契約書の不利条項・抜け・リスク箇所を抽出・解説 |
| [契約管理（締結・更新）](./契約管理（締結・更新）/) | 締結状況・有効期限・更新タイミングの管理とアラート |

## 前後のつながり

```
hearing.skill（商談）
      ↓
NDA作成・レビュー（情報共有前）
      ↓
業務委託契約書作成（受注時）← estimate.skill
      ↓
契約書リスクレビュー（相手方の契約確認）
      ↓
契約管理（締結後の管理）
```

## 優先度
- ★★★ 今すぐ：業務委託契約書作成、NDA作成・レビュー、契約書リスクレビュー
- ★★☆ 3ヶ月以内：利用規約・PP作成
- ★☆☆ 将来：契約管理（締結・更新）

## 注意事項
生成された契約書はあくまでドラフト。
重要な契約は必ず弁護士・司法書士に最終確認を依頼すること。
