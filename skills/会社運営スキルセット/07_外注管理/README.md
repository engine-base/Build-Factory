# 07_外注管理

**役割：** 外部リソース（フリーランサー・AIエージェント）を「チーム」として機能させる。
一人でも組織のように動くための仕組み。

---

## スキル一覧

| スキル | やること |
|--------|---------|
| [外注タスク設計](./外注タスク設計/) | 仕事の切り出し・スコープ定義・ブリーフ生成。distributed-dev.skillの外注人間版 |
| [外注先評価・選定](./外注先評価・選定/) | 候補者のポートフォリオ・スキルセットのスクリーニング基準設計と評価シート生成 |
| [外注進捗管理・検収](./外注進捗管理・検収/) | 納品物の品質チェックリスト生成・フィードバック文の自動生成 |
| [業務マニュアル（SOP）作成](./業務マニュアル（SOP）作成/) | 繰り返し発生する業務の標準手順書を生成し、誰でも再現できる形にする |

---

## 前後のつながり

```
task-decomposition.skill（タスク分解）← 開発OS
      ↓
外注タスク設計（外部に出す形に変換）
      ↓
外注先評価・選定 → 外注進捗管理・検収
      ↓
業務マニュアル（SOP）作成（次回以降の再利用）
```
