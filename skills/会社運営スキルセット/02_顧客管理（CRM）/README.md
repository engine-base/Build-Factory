# 02_顧客管理（CRM）

**役割：** 商談前後の関係構築と案件パイプラインの管理。
顧客との信頼を維持し、継続受注につなげる。

---

## スキル一覧

| スキル | やること |
|--------|---------|
| [案件パイプライン管理](./案件パイプライン管理/) | リード〜成約〜納品の状態遷移を可視化し、次のアクションを自動生成 |
| [顧客フォローアップ](./顧客フォローアップ/) | 定期連絡文の生成・フォローアップのタイミング設計 |
| [議事録・商談メモ](./議事録・商談メモ/) | 会話・録音→構造化議事録→次アクション抽出まで一気通貫 |
| [顧客レポート生成](./顧客レポート生成/) | 進捗・実績の定期レポートを自動作成してクライアントに送付 |
