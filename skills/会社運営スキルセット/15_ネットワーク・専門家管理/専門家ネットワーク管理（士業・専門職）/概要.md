# 専門家ネットワーク管理（士業・専門職）

## このスキルがやること
税理士・弁護士・社労士・デザイナー・エンジニア等の専門家との関係を一元管理し、
必要な時に即座に相談できる体制を整える。
「そういえば誰に聞けばいいかわからない」をなくす。

---

## 管理すべき専門家カテゴリ

| カテゴリ | 役割 | 優先度 |
|---------|------|--------|
| 税理士 | 確定申告・節税・記帳サポート | ★★★ |
| 弁護士 | 契約レビュー・法的トラブル対応 | ★★★ |
| 社会保険労務士 | 社会保険・労務管理（外注が増えたら） | ★★☆ |
| ファイナンシャルプランナー | 保険・資産設計 | ★★☆ |
| 同業デザイナー | デザインが必要な時の外注先 | ★★☆ |
| 同業エンジニア | 自分のスキル外の開発を依頼できる人 | ★★☆ |
| メンター・先輩経営者 | 経営判断の相談相手 | ★★★ |
