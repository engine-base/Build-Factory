# 15_ネットワーク・専門家管理

**役割：** 一人会社でも「チーム」を持つ。
税理士・弁護士・専門職との関係を管理し、紹介パートナーを育て、業界人脈を維持する。
「全部自分でやらなければならない」という思い込みを捨て、必要な時に適切な専門家を使える状態を作る。

---

## なぜ「人脈管理」にスキルが必要か

人脈は放置すると薄れる。
意図的にメンテナンスしないと「いざという時に頼れる人がいない」状態になる。
このカテゴリは「関係を管理する」のではなく「関係を意図的に育てる」ための設計。
