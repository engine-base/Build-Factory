# KPI設計・モニタリング

## このスキルがやること
事業フェーズに合ったKPIを定義し、追跡方法・ダッシュボード設計・アラート設定を行う。
「何を見れば会社の健康状態がわかるか」を明確にする。

## インプット
- 事業フェーズ（立ち上げ / 成長 / 安定）
- ビジネスモデル（受託 / SaaS / 物販等）
- 現在の課題・注力エリア

## アウトプット
- KPI一覧（北極星指標 + 先行指標 + 遅行指標）
- 各KPIの計測方法・データソース
- ダッシュボード設計（Notion / Googleスプレッドシート等）
- 週次・月次モニタリングルーティン
