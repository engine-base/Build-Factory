# 05_経営戦略

**役割：** 会社の方向性を決め、正しい意思決定を支援する。
「何をやるか・何をやらないか」の判断をデータと構造で支える。

---

## スキル一覧

| スキル | やること |
|--------|---------|
| [市場調査](./市場調査/) | 参入市場の規模・成長性・顧客セグメント・トレンドをリサーチして構造化レポート化 |
| [競合分析](./競合分析/) | 競合サービスの機能・価格・強み・弱みを比較表にまとめ、差別化ポイントを抽出 |
| [事業計画書作成](./事業計画書作成/) | ビジョン・戦略・数値計画（売上目標・コスト）を統合した事業計画書の生成 |
| [KPI設計・モニタリング](./KPI設計・モニタリング/) | 事業フェーズに合ったKPIの定義・トラッキング設計・ダッシュボード設計 |
| [意思決定記録](./意思決定記録/) | 重要な経営判断の背景・選択肢・決定理由を記録し、後から参照できる形で保存 |
| [投資家・銀行向け資料](./投資家・銀行向け資料/) | 融資・出資申請用の財務サマリ・事業説明資料の生成 |

---

## 前後のつながり

```

```
