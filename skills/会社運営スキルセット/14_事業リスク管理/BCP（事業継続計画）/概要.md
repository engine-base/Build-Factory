# BCP（事業継続計画）

## このスキルがやること
「自分が急病になった」「パソコンが壊れた」「自然災害が起きた」などの非常事態でも、
可能な限り事業を継続・復旧できる計画を設計する。
一人会社では「自分が止まる = 事業が止まる」リスクが最大。これを最小化する。

---

## リスクシナリオ別の対応計画

### シナリオ1：短期的な体調不良（1〜2週間）
```
事前対策：
□ 進行中の全案件に「バッファ」を2週間以上持たせる
□ 各クライアントへの緊急連絡先を決めておく（信頼できる同業者等）
□ 重要タスクの引き継ぎドキュメントを常時最新化（SOPスキルと連携）

発生時：
□ 即日：クライアントへの状況連絡（テンプレート準備済み）
□ 2日以内：納期調整の交渉（業務委託契約書の「不可抗力条項」確認）
```

### シナリオ2：機器障害（PC故障・データ消失）
```
事前対策：
□ 全データをクラウドに自動バックアップ（Google Drive / Dropbox等）
□ パスワード・アカウント情報を安全な場所に保管（1Password等）
□ 緊急用PCの確保手段を決める（家族のPC・コワーキングスペース等）

発生時：
□ 別端末で即日作業再開できる環境を設計
```

### シナリオ3：主要ツールのサービス停止
```
事前対策：
□ 重要ツールの代替候補を1つ決めておく
□ データのエクスポート・バックアップを定期実施

発生時：
□ 代替ツールで当日中に作業再開
```
