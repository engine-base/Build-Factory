# 14_事業リスク管理

**役割：** 一人会社の最大の弱点は「自分が止まると全てが止まる」こと。
売上・健康・ツール・情報・顧客の集中リスクを事前に把握し、継続できる仕組みを作る。
起きてから対応するのではなく「起きても大丈夫な状態を作る」ための守りのカテゴリ。

---

## スキル一覧

| スキル | やること |
|--------|---------|
| [BCP（事業継続計画）](./BCP（事業継続計画）/) | 自分が倒れた・災害が起きた時でも事業を継続する計画 |
| [顧客・売上依存リスク管理](./顧客・売上依存リスク管理/) | 特定顧客・特定業種への依存度を把握し分散計画を立てる |
| [保険管理](./保険管理/) | 賠償責任・所得補償・各種保険の加入状況を最適化する |
| [データ・セキュリティ管理](./データ・セキュリティ管理/) | バックアップ・アカウント管理・情報漏洩対策を設計する |

---

## リスクの優先度マトリクス

```
影響度が大 × 発生確率が高
  → 今すぐ対策：顧客集中リスク・体調管理・バックアップ

影響度が大 × 発生確率が低
  → 保険・BCPで備える：大規模災害・法的紛争

影響度が小 × 発生確率が高
  → ルーティン化：ツール障害・締め切りミス

影響度が小 × 発生確率が低
  → 気にしない
```
