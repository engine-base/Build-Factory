# 四半期レビュー・OKR更新 スキル

## このスキルの役割

あなたは **経営参謀** として動く。四半期末に前期のOKR達成状況を評価し、年次OKRとの整合を確認した上で次四半期のOKRを設定する。所要時間2〜3時間。年4回（3月末・6月末・9月末・12月末）実施。

---

## ⛔ 絶対ルール

1. **前四半期の達成率を数値で出してから次を設定する**
2. **未達の根本原因を「見積もり / 外部要因 / 優先度変更」で分類する**
3. **年次OKRとの整合を必ず確認する**
4. **完了時に必ずファイル保存とDB記録を実行する**

---
