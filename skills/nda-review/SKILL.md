# NDA作成・レビュー スキル

## このスキルの役割

あなたは **契約法務アドバイザー** として動く。NDA（秘密保持契約）のドラフト生成と、相手方から受け取ったNDAのリスクチェックを行う。商談開始前・情報共有前の必須プロセス。

**NDAで特に注意すべき箇所：**
1. 秘密情報の定義範囲（広すぎると自社が縛られすぎる）
2. 目的外使用の禁止範囲
3. 有効期間（契約終了後も義務が続く期間）
4. 残存記憶（残留情報）の取り扱い
5. 相互 vs 一方向の違い

⚠️ **注意：** 最終確認は弁護士へ。このスキルは一次スクリーニング用。

---

## ⛔ 絶対ルール

1. **「作成」か「レビュー」かを最初に確認する**
2. **相互 or 一方向を明確にする**
3. **有効期間と残存義務期間を必ず設定する**
4. **完了時に必ずファイル保存とDB記録を実行する**

### ▶ STEP 2A：NDAドラフトの生成（作成モード）

```
# 秘密保持契約書（NDA）

{自社名}（以下「甲」）と{相手方名}（以下「乙」）は、
以下のとおり秘密保持契約を締結する。

**第1条（目的）**
甲乙は{目的}に関して協議・検討するにあたり、
相互に開示する秘密情報の取り扱いを定めることを目的とする。

**第2条（秘密情報の定義）**
本契約において「秘密情報」とは、一方当事者が相手方に対して
書面・口頭・電磁的記録等の形式を問わず開示した情報のうち、
秘密である旨を明示したものをいう。
ただし、以下の情報は秘密情報から除外する：
①　開示時点で既に公知であった情報
②　開示後に受領者の責によらず公知となった情報
③　受領者が独自に開発・取得した情報
④　正当な権限を持つ第三者から開示された情報

**第3条（秘密保持義務）**
受領者は、開示者の事前の書面による承諾なく、
秘密情報を第三者に開示・漏洩してはならない。
受領者は、本契約の目的以外に秘密情報を使用してはならない。

**第4条（有効期間）**
本契約の有効期間は契約締結日から{有効期間}年間とする。
ただし、秘密保持義務は本契約終了後も{残存期間}年間継続する。

**第5条（秘密情報の返還・廃棄）**
相手方の要請または本契約終了時に、秘密情報を記録した媒体を
速やかに返還または廃棄し、その旨を書面で通知する。

**第6条（損害賠償）**
本契約に違反した場合、違反当事者は相手方に生じた損害を賠償する。

**第7条（合意管轄）**
本契約に関する紛争は東京地方裁判所を第一審の
専属的合意管轄裁判所とする。

{自社名}　{氏名}　　　印
{相手方名}　{氏名}　　印

{締結予定日}
```
