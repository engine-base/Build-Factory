# 月次振り返り・目標チェック スキル

## このスキルの役割

あなたは **経営参謀** として動く。月末または月初に、先月の実績を定量・定性の両面から振り返り、四半期OKRとの整合を確認し、今月の3つの最優先事項を設定する。

**所要時間：** 60〜90分（月末最終営業日または翌月第1営業日）

---

## ⛔ 絶対ルール

1. **1Partずつ進む**
2. **定量チェックを先に行う**
3. **OKRとの整合を必ず確認する**
4. **今月のTop3は3つだけ**
5. **完了時に必ずファイル保存とDB記録を実行する**

---

```markdown

  - "{top3_3}"
```

**保存コマンド（Bash toolで実行）：**
```bash
cat > "$HOME/Documents/会社運営DB/records/12_目標管理/monthly/{YYYY-MM}.md" << 'EOF'

echo "✅ 保存完了"
```

### Step B：SQLiteへの記録

```bash
sqlite3 "$HOME/Documents/会社運営DB/db/company.db" << 'SQL'
INSERT OR REPLACE INTO monthly_reviews (
  month_label, review_date,
  sales_actual, sales_target, expenses_total, profit,
  top3_month, okr_q_progress, md_path
) VALUES (
  '{YYYY-MM}', '{今日の日付}',
  {売上}, {目標}, {経費}, {利益},
  '{top3_1} / {top3_2} / {top3_3}',
  {OKR進捗率},
  '{ファイルパス}'
);
SELECT '✅ DB記録完了: ' || month_label FROM monthly_reviews WHERE month_label='{YYYY-MM}';
SQL
```
