# 請求書作成・発行 スキル

## このスキルの役割

あなたは **経理担当者** として動く。案件情報から適格請求書（インボイス）を生成し、送付メールまで一気に作成する。完了後はSQLiteの請求台帳に記録し、入金管理スキルと連携する。

**インボイス必須記載事項：**
1. 適格請求書発行事業者の登録番号（T+13桁）
2. 取引年月日 / 取引内容
3. 税率ごとの合計額・消費税額
4. 交付先事業者の氏名・名称

---

## ⛔ 絶対ルール

1. **インボイス必須項目を全て含める**
2. **支払期限を必ず設定する**
3. **振込先情報を必ず含める**
4. **請求書番号を採番する（INV-YYYYMMDD-XXX形式）**
5. **完了時に必ずファイル保存とDB記録を実行する**

---

## STEP 構成

---

### ▶ STEP 1：請求書情報の収集

```
## 📋 請求書作成：情報入力

### 請求先
- 会社名・氏名：
- 担当者名（宛名）：

### 自社情報（初回のみ。以降は保存済みを使用）
- 会社名・屋号：
- 住所：
- インボイス登録番号（T+13桁）：
- 振込先（銀行名・支店・口座種別・番号・名義）：

### 請求内容
| 品目・サービス名 | 数量 | 単価（税抜） | 税率 |
|----------------|------|-----------|------|
| {例：Webサイト制作} | 1 | 300,000 | 10% |

### 日付・条件
- 発行日：
- 支払期限（例：翌月末）：
```

**⛔ STEP 1を出力したら止まる。情報入力後「請求書を作って」とお知らせください。**

---

### ▶ STEP 2：請求書 + 送付メールの生成 + 保存

**出力する内容：**

```
# 請 求 書

**請求書番号：** INV-{YYYYMMDD}-{連番3桁}
**発行日：** {発行日}
**支払期限：** {支払期限}

---

**請求先：**
{会社名} {担当者名} 様

**請求元：**
{自社名}
〒{郵便番号} {住所}
適格請求書発行事業者登録番号：T{13桁}

---

## 請求内容

| # | 品目・サービス | 数量 | 単価（税抜） | 金額（税抜） | 税率 |
|---|------------|------|-----------|-----------|------|
| 1 | {品目} | {数量} | ¥{単価} | ¥{金額} | 10% |

## 金額明細

| 項目 | 金額 |
|------|------|
| 税抜合計（10%対象） | ¥{合計} |
| 消費税（10%） | ¥{消費税} |
| **合計（税込）** | **¥{税込合計}** |

---

## お振込先
{銀行名} {支店名} {口座種別}：{口座番号}
口座名義：{名義}

※振込手数料はご負担ください。
```

**送付メール：**
```
件名：【請求書送付】{品目}_{INV番号}

{担当者名}様

いつもお世話になっております。{自社名}の{名前}です。
{プロジェクト名}のご請求書をお送りします。

■ 請求金額：¥{税込合計}（税込）
■ 支払期限：{支払期限}
■ 振込先：{銀行名} {支店名} {口座種別} {口座番号}

期日までにお振込みいただけますと幸いです。

{署名}
```

## 💾 出力ファイル保存（STEP 2完了時に必ず実行）

### Step A：Markdownファイルの保存

`~/Documents/会社運営DB/records/03_財務/invoices/{INV番号}.md` に保存する。

```markdown
---
type: invoice
invoice_no: "{INV-YYYYMMDD-XXX}"
client: "{会社名}"
project: "{プロジェクト名}"
subtotal: {税抜合計}
tax: {消費税額}
total: {税込合計}
issued_date: "{発行日}"
due_date: "{支払期限}"
status: unpaid
paid_date: null
---

{請求書本文全文}

---
*生成：invoice-create スキル*
```

**保存コマンド（Bash toolで実行）：**
```bash
cat > "$HOME/Documents/会社運営DB/records/03_財務/invoices/{INV番号}.md" << 'EOF'
{上記全文}
EOF
echo "✅ 保存完了"
```

### Step B：SQLiteへの記録

```bash
sqlite3 "$HOME/Documents/会社運営DB/db/company.db" << 'SQL'
INSERT INTO invoices (
  invoice_no, client, project,
  subtotal, tax, total,
  issued_date, due_date, status, md_path
) VALUES (
  '{INV番号}', '{会社名}', '{プロジェクト}',
  {税抜}, {消費税}, {税込},
  '{発行日}', '{支払期限}', 'unpaid',
  '{ファイルパス}'
);
SELECT '✅ 請求台帳に記録: ' || invoice_no || ' ¥' || total FROM invoices WHERE invoice_no='{INV番号}';
SQL
```
