RLS Best Practices for Build-Factory

原則

  • 全 entity に 必ず RLS policy を当てる (entities.json declared = 43, 実装も 43 必須)
  • workspace 系: workspace_id = current_workspace_id + member 確認
  • account 系: account_id = current_account_id + role 確認

PostgreSQL session GUC で current_user 管理

CREATE POLICY tasks_select_member ON tasks
  FOR SELECT USING (
    workspace_id IN (
      SELECT workspace_id FROM workspace_members
      WHERE user_id = current_setting('app.current_user_id')::uuid
    )
  );

FastAPI dependency で GUC 設定

require_auth dependency で SET LOCAL app.current_user_id = $jwt.sub を Postgres connection に流す。
