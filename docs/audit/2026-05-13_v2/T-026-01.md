### Skip-with-reason

- pytest 経由で **本物の Postgres に接続** する統合テストは Phase 1 sandbox 範囲外 (sqlite 環境)。 → 静的 SQL invariant + service 単体 (sqlite で _Conn mock 経由) で代替。理由 1 行: 「Postgres 統合は T-001-11 db_integration が網羅 / 本 PR は spec rigor」.
- DDL 変更 (max_size_kb 強制 CHECK 追加) は ADR が必要なため本 PR では追加しない。 理由 1 行: 「F-026 policies は loader/UI 側で計測 / DDL 強制は別 ADR」.
